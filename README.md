# Spring board

A simple bulletin board built with Spring Boot 2.

* Login with username and password
* Provides version control for posts and comments (in dev)

## How to run
```
$ ./gradlew bootJar
$ java -jar build/libs/spring-board-0.0.1-SNAPSHOT.jar
```
or you can just...
```
$ ./gradlew bootRun
```
